# Discord Mass Reporter
https://venumzmail.xyz

A tool that sends reports to Discord's API to report users or servers in bulk.

## Features

- Report users and servers
- Multiple report reasons (harassment, spam, nsfw, impersonation, self harm, copyright, illegal content)
- Proxy support
- Multi-threading for fast reporting
- Random user agents
- Console title shows progress
- Gradient colored output

## Requirements

- Python 3.8 or higher
- Discord tokens
